# Программа, выполненная на 8 баллов, для побайтного копирования файлов с использованием только системных вызовов ОС (Дз 7)

## Исполнитель
- **ФИО:** Чекмарев Трофим Владимирович
- **Группа:** БПИ-245

## Краткое описание программы
Программа предназначена для побайтного копирования файлов (текстовых и бинарных, включая исполняемые) с использованием только системных вызовов ОС. Имена исходного и целевого файлов передаются через аргументы командной строки. Для чтения и записи применяется циклический буфер фиксированного размера 32 байта, который поочерёдно заполняется данными из входного файла и опустошается при записи в выходной файл. Перед началом копирования программа получает с помощью fstat права доступа исходного файла и после завершения записи применяет их к выходному файлу через fchmod. Таким образом, если исходный файл был исполняемым (биты x установлены), то и его копия сохраняет возможность запуска, а обычные текстовые файлы, не имеющие признака исполнимости, после копирования исполняемыми не становятся.
